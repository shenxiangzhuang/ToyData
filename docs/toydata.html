class toydata.Queue.ArrayDeque[source]¶

Bases: toydata.Queue.ArrayQueue

Double-Ended Queues implementation using Python list as unedrlying storage.

[Michael T. Goodrich]The efficiency of an ArrayDeque is similar to that of an ArrayQueue, with all operations having O(1) running time, but with that bound being amortized for operations that may change the size if the underlying list.
